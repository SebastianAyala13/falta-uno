Expo's APIs change between SDKs. This project is on Expo SDK 57 (see `expo` in `package.json`). Before writing any Expo/React Native code, read the exact versioned docs for the installed SDK:

https://docs.expo.dev/versions/v57.0.0/

Do not rely only on `/versions/latest/` when implementing APIs for this project. Always use the versioned SDK docs that match the installed Expo version in `package.json`.

If you bump the Expo SDK, update this version and link too.

See `CLAUDE.md` for architecture, the pnpm/Node setup, and web-export gotchas.

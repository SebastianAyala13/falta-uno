# Expo: read the versioned docs first

Expo's APIs change between SDKs. This project is on **Expo SDK 54** (see `expo` in
`package.json`). Before writing any Expo/React Native code, read the exact versioned
docs for the **installed** SDK:

https://docs.expo.dev/versions/v54.0.0/

If you bump the Expo SDK, update this version and link too.

See `CLAUDE.md` for architecture, the pnpm/Node setup, and web-export gotchas.
